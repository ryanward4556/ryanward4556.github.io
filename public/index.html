Job Trek (group project #2)
  • Motivation: If you are planning a trip in another city, or just want something to do this weekend, Event Finder is here to help
  • Description: This application will help you plan events for your next trip. Using the TicketMaster and Eventful API's, we take the date range and the city you select and find all of the events pertaining to your dates and and city. Each search is saved to a Firebase database, which can be used for future improvements to save searches for different users
  • Technologies used:
    • HTML / CSS
    • Bootstrap
    • JQuery
    • Sass
    • Bounce.js (animations)
    • Node
    • Express
    • Sequelize / MySQL
    • Passport (Local Authentication)
    • AJAX
  • API's used:
    • Authetic Jobs
